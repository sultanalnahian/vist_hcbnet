# HCBNet: Hierarchical Context-based Network

This repository is the implementation of [A Hierarchical Approach for Visual Storytelling Using Image Description](https://arxiv.org/pdf/1909.12401.pdf) 
<br>
## Prerequisites:
1. Python 3.6
2. pytorch >= 1.0.0

<br>
### Training
```
python train.py
```
